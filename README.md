# Jarvis – Autonomous AI Assistant

Jarvis is a multi-agent AI assistant built using n8n and LLMs to automate productivity workflows such as email management, calendar scheduling, and contact retrieval.

## Features

- Send professional emails automatically
- Reply within existing email threads
- Schedule meetings via Google Calendar
- Retrieve contacts from Google Sheets
- Telegram interface for commands
- Multi-agent architecture

## Architecture

Jarvis uses a central orchestrator that coordinates:

- Email Agent
- Calendar Agent
- Contact Agent

## Tech Stack

- n8n
- LLM (OpenAI / Gemini)
- Gmail API
- Google Calendar API
- Google Sheets
- Telegram Bot API
